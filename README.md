# Digital Literacy Portfolio

## Project Overview
This repository contains my **Digital Literacy Portfolio** developed as part of the **CSE0001 – Digital Literacy** course at VIT Bhopal University.

As a *Student Digital Ambassador*, this project focuses on building practical skills required to navigate the digital world effectively, including creating professional profiles, using collaborative tools, practicing safe online behavior, and understanding cyber threats.

The project is structured into five tasks, each aligned with a specific course module. It demonstrates hands-on learning using real-world platforms and tools, along with reflections on their applications in academic and professional contexts.

## Module-wise Summary

### Module 1: Presentation & Digital Awareness
This module focuses on creating visual content using tools like Canva, Prezi, and PowerPoint. It introduces the fundamentals of digital literacy, including safe internet practices, professional presence, and effective communication through visuals.

**Application in Project:**  
Designed a digital literacy infographic highlighting key concepts such as online safety, digital tools, and responsible usage, using Canva as the main tool for development 

---

### Module 2: Digital Portfolio Development
This module emphasizes building a professional online presence using platforms such as GitHub, LinkedIn, and Kaggle. It highlights the importance of maintaining a digital identity for academic and career growth.

**Application in Project:**  
Created and updated profiles across selected (Github, LinkedIn and Kaggle) platforms and structured a professional portfolio for long-term use.

---

### Module 3: Coding & Collaboration Platforms
This module introduces coding practice platforms (HackerRank, CodeChef) and collaboration tools (Google Docs, Forms, Sheets). It develops familiarity with cloud-based workflows and problem-solving environments.

**Application in Project:**  
- Completed the Python Introduction Challenge in HackerRank 
- Created a Google Form quiz for digital literacy awareness 

---

### Module 4: Email & Social Media Etiquette
This module covers professional communication standards, including email formatting, tone, structure, and responsible social media usage.

**Application in Project:**  
- Drafted professional emails for assignment deadline extension and for showcasing interest in an internship opportunity
- Created a checklist of Do’s and Don’ts for social media behavior  

---

### Module 5: Cybercrime Awareness & Prevention
This module provides awareness about cyber threats such as phishing, identity theft, and financial fraud. It also emphasizes prevention strategies and digital hygiene.

**Application in Project:**  
- Developed a case study on Ransomware, particularly on Wannacry ransomware attack of 2017
- Created a “Stay Safe Online” prevention checklist with actionable tips
